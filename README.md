
# Customer Retention & Churn Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes customer retention and churn behavior using the Telco Customer Churn dataset.

## Dashboard Features
- Total Monthly Charges KPI
- Total Charges KPI
- Total Tenure KPI
- Customer Distribution by Gender
- Internet Service Analysis
- Churn Distribution
- Contract Type Analysis
- Payment Method vs Churn
- Business Insights

## Tools Used
- Power BI
- Data Visualization
- Business Analytics

## Key Insights
- Churn rate is approximately 26.5%.
- Most customers are retained.
- Fiber Optic is the most used internet service.
- Electronic Check customers show higher churn.
- Long-term contracts contribute higher revenue.
